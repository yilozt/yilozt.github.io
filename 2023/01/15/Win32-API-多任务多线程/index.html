Win32 API 多任务多线程

yilozt
2023-01-15
Win32

头文件：

#include <process.h>

  • 如何创建线程？CreateThread
    简便一点的 API：_beginthread

  • 进程同步
    临界区：CRITICAL_SECTION
    进入临界区：EnterCriticalSection
    离开临界区：LeaveCriticalSection
    初始化： InitializeCriticalSection
    释放资源：DeleteCriticalSection

  • 事件对象 可以看成是一个特殊的 bool 变量，但是可以阻塞线程
    CreateEvent
    WaitForSingleObject

  • TLS 线程区域存储空间
    线程共享全局变量？如果需要给线程设置不同的值：TLS

    可以理解为一块特殊的内存区域，在里面定义的全局变量后，每个线程都可以给变量设置自己的值

    _declspec (thread) int a = 0;
    _declspec (thread) static int a = 0;

    TlsAlloc() // 分配索引
    TlsSetValue() // 为索引设置存储的内容
    TlsGetValue() // 获取内容
    GlobalFree() // 释放空间
    TlsFree() // 回收索引
